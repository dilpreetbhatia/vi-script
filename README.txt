README for vi-script 


FILES INCLUDED
--------------
index.html
controller.js
hdFlagsProcessor.php
default.css


SETTING UP THE PROJECT
----------------------
You will need PHP supported Apache server installation. I used XAMPP.
Create a new project folder in htdocs directory and copy the files mentioned above in that. 
All the 4 files should be in the same directory.



RUNNING THE PROJECT
-------------------
Load the index.html. 
Click 'Lets Start' Button
The running count of the HD flags will be displayed
Once the more=false flag is encountered, script will display the final count of hd flags.

LOGIC
-----
Comments in hdFlagsProcessor.php explain the logic used in more detail.
